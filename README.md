# 🤖 AI-Powered Career Guidance & Job Apply Agent
**IBM SkillsBuild AI Strategy & Business Intelligence Internship Final Project**

## 🌟 Project Overview
This Agentic AI assistant bridges the employment gap for graduates in Tier-2 and Tier-3 cities in India by automating the job application lifecycle. [cite_start]It uses advanced AI to parse resumes, match candidates with relevant roles, and automate professional outreach. [cite: 61, 62, 63, 64]

### 🎯 Sustainable Development Goals (SDG) Alignment
- [cite_start]**SDG 4 (Quality Education):** Providing high-end career guidance to underserved regions. [cite: 66, 86, 160]
- [cite_start]**SDG 8 (Decent Work & Economic Growth):** Optimizing the transition from education to employment. [cite: 66, 161]

## 🛠️ Technical Architecture & Stack
[cite_start]The system is built using a modern full-stack architecture optimized for high-performance agentic workflows: [cite: 172, 173]

- [cite_start]**Frontend:** React.js (Vite) + Tailwind CSS for a responsive and minimalist user interface. [cite: 174, 186]
- [cite_start]**Backend:** Python + FastAPI for high-performance logic and API management. [cite: 175]
- [cite_start]**Orchestration:** **LangGraph** multi-step agent orchestration to ensure reliable state transitions. [cite: 176, 178]
- [cite_start]**Model:** Llama 3.3 (via Groq API) for fast, accurate resume parsing and letter generation. [cite: 177]


## 🚀 Key Features
1. [cite_start]**Automated Resume Parsing:** Extracts skills and experience using Named Entity Recognition (NER). [cite: 68, 78, 199]
2. [cite_start]**Dynamic Job Matching:** Aligns candidate profiles with market opportunities in real-time. [cite: 69, 79, 167, 202]
3. [cite_start]**AI Cover Letter Generation:** Drafts role-specific, professional cover letters tailored to the user's background. [cite: 70, 80, 227, 246]
4. [cite_start]**Smart Dispatch:** Direct Gmail integration for immediate HR outreach, reducing application time by up to 90%. [cite: 71, 73, 81, 272]
