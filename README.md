# 🤖 AI-Powered Career Guidance & Job Apply Agent
**IBM SkillsBuild AI Strategy & Business Intelligence Internship Final Project**

## 🌟 Project Overview
This Agentic AI assistant bridges the employment gap for graduates in Tier-2 and Tier-3 cities in India by automating the job application lifecycle. It simplifies resume parsing, job matching, and professional outreach into a single, intuitive workflow, reducing manual effort by up to 90%.

### 🎯 Sustainable Development Goals (SDG) Alignment
- **SDG 4 (Quality Education):** Ensuring skill-based career growth and democratization of career guidance.
- **SDG 8 (Decent Work & Economic Growth):** Optimizing the transition from education to employment through AI automation.

## 🛠️ Technical Architecture & Stack
The system is built using a modern full-stack architecture optimized for high-performance agentic workflows:

- **Frontend:** React.js (Vite) + Tailwind CSS for a responsive and minimalist user interface.
- **Backend:** Python + FastAPI for high-performance logic and API management.
- **AI Orchestration:** **LangGraph** multi-step agent orchestration to ensure reliable state transitions between processing nodes.
- **Model:** Llama 3.3 (via Groq API) for fast, accurate reasoning and natural language generation.

## 🚀 Key Features
1. **Automated Resume Parsing:** Uses AI-powered Named Entity Recognition (NER) to extract candidate names, roles, and technical skills from PDF resumes.
2. **Dynamic Job Matching:** Aligns candidate profiles with current market opportunities in real-time based on extracted skill sets.
3. **AI Cover Letter Generation:** Drafts role-specific, professional business cover letters tailored to the user's specific background and the target job description.
4. **Smart Dispatch:** Direct Gmail integration for immediate HR outreach, pre-filling recipient details, subject lines, and body content for seamless communication.
